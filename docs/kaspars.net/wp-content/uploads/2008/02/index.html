Getting it consistent among browsers

Notice: All of the following examples use an additional CSS identifier ul.normal and ol.normal, which for you should be simply ul and ol.

CSS:

/* Used by IE7, but wont't 
   harm anybody else. */
body { min-height:0; }

ul, ol { 
   margin:auto -3em 1em 0; 
   padding:0; 
   position:relative; 
   left:-3em;
   overflow:hidden;
}

ul ul, ul ol, 
ol ol, ol ul { 
   margin-left:1em; 
   padding-left:0; 
}

ul li, ol li { 
   margin-left:5em; 
}

li li { 
   margin-left:1em; 
}

ul, ol { 
   /* Apply only (!) 
      to IE6 */
   height:0;
   overflow:visible;
}
